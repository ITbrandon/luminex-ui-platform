# Luminex UI Platform

A modern, enterprise-grade **UI component library** built with **Lit.js Web Components** and demonstrated in a **React + Vite** dashboard. Luminex is designed to be **framework-agnostic, visually vibrant, accessible**, and **reusable across multiple internal applications**.
